**What.** `/dream` = hippo's GENERATIVE sleep pass — the one function of sleep the housekeeping verbs don't cover. `consolidate`/`reconsolidate`/`staleness`/salience already do systems-consolidation, reconsolidation, decay. `/dream` does REPLAY that RECOMBINES existing memories into net-new structure: an offline turn that re-activates the corpus against itself, watches what co-fires, and proposes the latent graph edges the corpus is structurally missing (transitive A–B–C bridges, named-but-unlinked in a body, undeclared typed relations) — the edge class `consolidate`'s two signals (write-time similarity GRA-3 + co-recall GRW-2) structurally cannot reach. Those edges are what `recall._expand_neighbors` (GRA-1) traverses to turn a 2-hop miss into a 1-hop hit.

**The core reframe (Fred's steer): REVERSIBLE AUTONOMY, not propose-only.** The safety mechanism is not human approval — it's that memory lives in git, so an additive edge is trivially reversible. Loop = `apply-reversibly → notify → undo-window → age-in`. This is the safe+additive layer on top of ~100% HITL Fred wanted. Three pillars: (1) reversibility gradient — Tier A (wikilink/dangling-completion/`refines` edges) AUTO-APPLY with digest+undo; Tier B (schemas/gist) auto-stage at `confidence:draft`, down-weighted, self-decaying; Tier C (`supersedes`/`contradicts`/hypotheses-as-verified) stay GATED. (2) The aging firewall — auto-applied edges influence recall immediately but are excluded from /dream's OWN source set until they age in (≥5 sessions or explicit bless) → kills the dream-cites-a-dream tower without a human. (3) notify-with-undo via real seams: MCP `_tool_dream` return, a `dream_applied_producer` SessionStart nudge, and git-native `dream --undo/--undo-since/--log`; effect is immediate (working tree, live in recall) but commit stays human.

**Proof.** Falsifiable: on a frozen snapshot, admitted edges raise `eval_recall by_category['multi-hop']['recall']@10` (the "validates GRA-1 expansion" bucket) BECAUSE of the edges — attribution via MuSiQue-style matched single-hop control (must stay flat). Guardrails (GEM-style): self_recall≥0.90 + zero hit→miss flips, abstention non-decreasing, precision@10≥0.12, net-token>0, MRR≥0.60, p95<300ms. Auto-apply's payoff: stamped edges let the `HIPPO_DREAM` A/B run on the LIVE corpus, not just a fixture.
